
#### disable 多个Table
``` bash
Ex: table name

nice.good.20161101
nice.good.20161102
nice.good.20161201

disable_all 'nice.good.201611.*'
will disable 2 tables
````



#### op 多个Table
``` bash
Ex: table name

nice.good.20161101
nice.good.20161102
nice.good.20161201

drop_all 'nice.good.201611.*'
will disable 2 tables
````
